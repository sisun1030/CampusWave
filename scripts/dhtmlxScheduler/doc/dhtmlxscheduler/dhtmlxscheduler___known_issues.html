Known issues

Problems, which are described below, are related to the scheduler 2.3

Problem: in case of Brazil timezone, during summer time shift, scheduler can render double days ( so far, there was only few reports from users of Brazil-version of Windows, problem doesn't appear for any other type of OS )
Symptom: double days in month view
Affected versions: all versions
Patch: http://support.dhtmlx.com/x-files/scheduler_2_1/brazil_patch.zip
Patch need to be included on the page after other js files.
