linkCalendar(calendar, shift)

Required library edition:
Required library file: dhtmlxscheduler_minical.js

After such command, each time when date is changed in scheduler , it will be changed in mini-calendar as well

Parameters:

  • calendar - calendar which was created with renderCalendar call
  • shift - allows to define how date in mini-calendar will differ from one in scheduler
	var calendar = scheduler.renderCalendar(...);
	scheduler.linkCalendar(calendar);
	//next month
	scheduler.linkCalendar(calendar, function(date){
		return scheduler.date.add(date, 1, "month");
	});
