onMouseMove Event

Parameters:

  • id of event;
  • native event object.
  scheduler.attachEvent("onMouseMove", function (event_id, native_event_object){
       //any custom logic here
  });

Event called when mouse hovered other scheduler. event_id parameter may be empty, if there is no event under the mouse pointer.
