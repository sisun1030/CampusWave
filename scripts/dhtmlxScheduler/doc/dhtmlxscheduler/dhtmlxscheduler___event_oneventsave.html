onEventSave Event

onEventSave event can call a user-defined handler (if there is any) and pass the following parameters:

  • id of event;
  • hash of event's data
  • flag of new event

Event is blockable and can be used for validation 

	scheduler.attachEvent("onEventSave",function(id,data,is_new_event){
		if (!data.text) {
			alert("Text must not be empty");
			return false;
		}
		if (!data.text.length<20) {
			alert("Text too small");
			return false;
		}
		return true;
	})
