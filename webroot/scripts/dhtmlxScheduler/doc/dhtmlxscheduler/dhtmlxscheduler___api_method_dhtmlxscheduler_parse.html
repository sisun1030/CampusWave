parse(data, type)

Required library edition:
Required library file: dhtmlxscheduler.js

load data from client side resource

  • url - string or object , which represents data
  • type - data type, can be “json” for loading from json object
   scheduler.parse("<data>...</data>");
   scheduler.parse([
	{ start_date:"2009-7-1 6:00", end_date:"2009-7-1 8:00", text:"Event 1"},
	{ start_date:"2009-7-2 6:00", end_date:"2009-7-2 8:00", text:"Event 2"}
   ],"json");
