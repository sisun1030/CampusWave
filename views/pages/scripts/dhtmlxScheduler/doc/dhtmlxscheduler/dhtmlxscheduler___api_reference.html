dhtmlxScheduler API Methods

Core

  • addEvent - adds new event
  • attachEvent - attaches an event handler to Scheduler
  • changeEventId - changes event id
  • clearAll - clears all data in scheduler.
  • deleteEvent - deletes existing event
  • detachEvent - removes an event handler
  • editStop - switches from editing mode to normal mode, disable editor if it's open
  • edit - edits event
  • endLightbox - used as part of custom lightbox form implementation
  • getEventEndDate - returns end date of event
  • getEventStartDate - returns start day of event
  • getEventText - returns event text
  • getEvent - returns event object by id
  • getEvents - returns collection of events, which occurs in defined period
  • getLabel - returns label text by option's id and property name (ver. 2.3+)
  • getState - returns details about current state of the scheduler (ver. 2.3+)
  • init - init scheduler
  • load - loads data in scheduler
  • parse - load self from JSON object or XML string
  • select - selects event
  • setCurrentView - sets the current mode and displayed date
  • setEventEndDate - sets end date of event
  • setEventStartDate - sets start date of event
  • setEventText - sets event text
  • setLoadMode - sets auto-loading mode
  • setLightboxSize - update size of details form
  • showLightbox - shows details form for the event
  • startLightbox - used as part of custom lightbox form implementation
  • resetLightbox - forces lightbox form re-creation
  • uid - returns unique ID ( unique inside current scheduler, not GUID )
  • unselect - unselects event
  • updateEvent - forces event updating

Units view

  • createUnitsView - create units view

Timeline view

  • createTimelineView - create timeline view

Tree-Timeline view

  • openSection - opens specified section (ver. 2.3+)
  • closeSection - closes specified section (ver. 2.3+)
  • openAllSections - opens all sections (ver. 2.3+)
  • closeAllSections - closes all sections (ver. 2.3+)
  • addSection - adds new section to specified parent section (ver. 2.3+)
  • deleteSection - deletes specified section (ver. 2.3+)
  • getSection - returns section object by id (ver. 2.3+)

Expand behavior

  • expand - switch to full screen mode
  • collapse - collapse back to normal size

Mini-calendar extension

  • renderCalendar - crate mini-cal
  • destroyCalendar - destroy mini-cal
  • isCalendarVisible - check is popup minical visible
  • markCalendar - mark some date in mini-cal
  • unmarkCalendar - unmark some date in mini-cal
  • updateCalendar - updates minicalendar view (ver. 2.3+)
  • linkCalendar - links minicalendar to scheduler (ver. 2.3+)

Serialize extension

  • toPDF - convert to the PDF
  • toXML - serialize self to XML string
  • toJSON - serialize self to JSON string
  • toICal - serialize sefl to iCal format

Recurring events extension

  • getRecDates - returns all occurrences of a recurring event (ver. 2.3+)
